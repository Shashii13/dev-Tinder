# 💻 DevTinder

A full-stack developer networking platform inspired by Tinder — built using the MERN stack.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based)
- 👤 Profile Management
- 🔎 Developer Feed
- ❤️ Send Connection Requests
- 🔄 Accept / Reject Requests (coming soon)

---

## 🛠️ Tech Stack

### Frontend
- React
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
